## Image Creation Guidelines

### Design Principles
- Use PalC Networks brand colors (primary blue #0041C2, cyan #00C2FF)
- Maintain consistent visual style across all images
- Include subtle network/tech elements (circuits, nodes, data flows)
- Professional, enterprise-grade aesthetic
- Dark backgrounds with blue/cyan accents work well

### Technical Requirements
- Resolution: 1200x675px minimum (16:9 aspect ratio)
- Format: PNG with transparency support
- File size: Optimize for web (under 500KB recommended)
- Color space: RGB

### Content Themes
- **Networking**: Switches, routers, network topologies, data flows
- **AI/ML**: GPU clusters, neural networks, data processing
- **Cloud**: Cloud providers, multi-cloud connectivity, hybrid architectures
- **Security**: Zero trust, segmentation, identity verification
- **Automation**: Pipelines, workflows, CI/CD, GitOps

- All images should be created and placed in their respective directories
- Images will be used as featured images on resource landing pages
- Ensure images are optimized for web performance
