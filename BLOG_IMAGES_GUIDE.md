## Image Specifications

- **Format:** PNG (recommended) or JPG
- **Recommended Dimensions:** 1200x675px (16:9 aspect ratio) or larger
- **File Size:** Optimize for web (under 500KB recommended)
- **Naming Convention:** Use lowercase with hyphens (kebab-case)

## Notes

- All images should be placed in the `public/images/blog/` directory
- The file names must match exactly (case-sensitive)
- Images will be automatically served with the basePath (`/palc-staging`) prefix on staging server
- If an image is missing, the system will fallback to a placeholder image
